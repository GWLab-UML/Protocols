1. Create new folder labeled "Images"
2. Open settings 
3. Navigate to "Files and Links" tab 
4. In the first section, change "Default location for new attachments" to "In the folder specified below"
5. Change "Attachment folder path" to the path of the "Images" folder you created in step 1
	- If you type "Images" into the textbox, the folder's path should  appear in the dropdown
6. Scroll down to "Links" section
7. Change "New link format" to "Path from current file"
8. Turn off "Use Wikilinks"
